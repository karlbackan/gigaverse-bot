# Gigaverse Token Extractor Extension

A simple Chrome extension to extract auth tokens from Gigaverse.io for use with the bot.

## Installation (Takes 30 seconds!)

1. **Install in Chrome**
   - Open Chrome and go to: `chrome://extensions/`
   - Turn ON "Developer mode" (toggle in top right)
   - Click "Load unpacked"
   - Select the `gigaverse-token-extension` folder
   - Done! The extension is now installed

## How to Use

1. Go to [gigaverse.io](https://gigaverse.io) and log in
2. Click the extension icon (puzzle piece) in your toolbar
3. Click "Copy Token" or "Copy as .env"
4. Paste into your bot's `.env` file

## Features

- Shows account info (username, wallet address)
- Shows token expiration date
- Copy token only or as `.env` format
- Works on any Gigaverse.io page
- No permissions required except for gigaverse.io

## Uninstall

To remove: Go to `chrome://extensions/` and click "Remove" on the extension.
